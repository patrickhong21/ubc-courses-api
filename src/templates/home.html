API Endpoints

Here are some example API endpoints. Of course, you can vary the URL param arguments.

  • Endpoint (GET):

    http://ubccourses.patrickhong.ca/api/v1/course/info

  • Endpoint (GET):

    http://ubccourses.patrickhong.ca/api/v1/course/info?dept=CPSC

  • Endpoint (GET):

    http://ubccourses.patrickhong.ca/api/v1/course/info?dept=CPSC&num=213&year=2023&session=W

  • Endpoint (GET):

    http://ubccourses.patrickhong.ca/api/v1/course/info?dept=CPSC&num=213§ion=101&year=2023&session=W
